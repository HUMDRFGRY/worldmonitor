World Monitor 将 344 个精选新闻源、49 个交互式地图图层和 AI 驱动分析整合为一个态势感知平台。它基于同一份代码库运行五个专门变体，分别面向不同领域：地缘政治、技术、金融、大宗商品以及积极全球趋势。

这个平台面向记者、安全分析师、研究人员，以及任何需要了解新闻标题之外世界正在发生什么的人。

## 你可以做什么

- **实时监控全球事件**：在 3D 地球仪或平面地图上查看 49 个可切换数据图层，覆盖军机、海上船舶、卫星、地震、野火、网络威胁等
- **阅读 AI 生成的情报简报**：将数百条新闻标题综合为可执行摘要，并附带来源归属与置信度评分
- **追踪国家稳定性**：通过国家不稳定指数（CII），使用冲突数据、社会动荡指标和新闻速度，实时评估 24 个国家
- **分析金融信号**：包括市场数据、预测市场、央行利率、大宗商品价格和海湾经济指标
- **完全在浏览器中运行**：可选的 ONNX Runtime Web 离线 AI 能力，让你的数据保留在本机
