World Monitor 基于同一份代码库运行五个专门变体，每个变体都针对不同的监控需求进行了优化。

## 平台变体

| 变体 | URL | 重点 |
|---------|-----|-------|
| **🌍 World Monitor** | [worldmonitor.app](https://worldmonitor.app) | 地缘政治情报、军事追踪、冲突监控、基础设施安全 |
| **💻 Tech Monitor** | [tech.worldmonitor.app](https://tech.worldmonitor.app) | 技术行业情报、AI/创业生态、云基础设施、科技事件 |
| **😊 Happy Monitor** | [happy.worldmonitor.app](https://happy.worldmonitor.app) | 精选正向新闻、全球进步追踪、科学突破、保护成果 |
| **💰 Finance Monitor** | [finance.worldmonitor.app](https://finance.worldmonitor.app) | 全球市场、证券交易所、央行、大宗商品、外汇、加密货币、经济指标 |
| **⛏️ Commodity Monitor** | [commodity.worldmonitor.app](https://commodity.worldmonitor.app) | 大宗商品市场、矿区、加工厂、供应链、贸易流动 |

顶部栏中的紧凑型 **变体切换器** 可在各变体之间无缝切换，同时保留你的地图位置和面板配置。

## World Monitor（地缘政治）

主变体聚焦于地缘政治情报、军事追踪和基础设施安全监控。

### 主要能力

- **冲突监控** - 实时追踪战争区、热点和危机区域的升级态势
- **军事情报** - 226 个军事基地、飞行追踪、海军舰船监控、突发检测
- **基础设施安全** - 海底电缆、管道、数据中心、互联网中断
- **经济情报** - FRED 指标、石油分析、政府支出、制裁追踪
- **自然灾害** - 地震、恶劣天气、NASA EONET 事件（野火、火山、洪水）
- **AI 分析** - 焦点点检测（AI 识别的新闻收敛区域，区别于静态情报热点）、国家不稳定评分、基础设施级联分析

### 情报面板

| 面板 | 用途 |
|-------|---------|
| **AI Insights** | LLM 合成的世界简报与焦点点检测 |
| **AI Strategic Posture** | 战区级军事力量聚合与打击能力评估 |
| **Country Instability Index** | 24 个受监控国家的实时稳定性评分 |
| **Strategic Risk Overview** | 结合所有情报模块的综合风险评分 |
| **Infrastructure Cascade** | 对电缆、管道和 chokepoint 的依赖分析 |
| **Live Intelligence** | 基于 GDELT 的主题 feed（军事、网络、核能、制裁） |

### 新闻覆盖

覆盖地缘政治、防务、能源、智库和地区新闻（中东、非洲、拉美、亚太）的 344 个精选来源。

## Tech Monitor

技术变体（[tech.worldmonitor.app](https://tech.worldmonitor.app)）提供面向技术行业监控的专门图层。

### 技术生态图层

| 图层 | 描述 |
|-------|-------------|
| **Tech HQs** | 主要科技公司的总部（Big Tech、独角兽、上市公司） |
| **Startup Hubs** | 主要创业生态，包含生态等级、融资数据和代表性公司 |
| **Cloud Regions** | AWS、Azure、GCP 数据中心区域及可用区数量 |
| **Accelerators** | Y Combinator、Techstars、500 Startups 和区域加速器位置 |
| **Tech Events** | 即将举行的大会和科技活动，带倒计时 |

### 技术基础设施图层

| 图层 | 描述 |
|-------|-------------|
| **AI Datacenters** | 来自 Epoch AI 数据集的 313 个 AI 计算集群 |
| **Undersea Cables** | 对云连接至关重要的海底光缆路线 |
| **Internet Outages** | 影响技术运营的网络中断 |

### 技术新闻分类

- **初创与 VC** - 融资轮、并购、创业新闻
- **网络安全** - 安全漏洞、入侵、威胁情报
- **云与基础设施** - AWS、Azure、GCP 公告、故障
- **硬件与芯片** - 半导体、AI 加速器、制造
- **开发者与开源** - 语言、框架、开源项目
- **技术政策** - 监管、反垄断、数字治理
