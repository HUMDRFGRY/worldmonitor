## AGPL-3.0 的含义

AGPL 是一种面向网络软件的强 copyleft 许可证。它在 GPL 的基础上增加了一条要求：如果你把修改后的 World Monitor 作为公开可访问的服务运行，你必须向该服务的用户提供你的源代码。

| 你可以 | 你必须 | 你不能 |
|---------|----------|------------|
| 查看、研究和学习源代码 | 以任何二进制分发方式同时提供源码 | 在没有商业许可证的情况下商业使用 World Monitor |
| 为个人/研究用途修改并创建衍生作品 | 将衍生作品以 AGPL-3.0 许可发布 | 以不同许可证再授权 |
| 为非商业用途运行个人实例 | 说明你对代码做了哪些修改 | 移除或遮蔽原始版权和许可证声明 |
| 把改进回馈给项目 | 向任何公开部署的用户提供源码访问 | 重新品牌并把它当成自己的产品出售 |

## 禁止重新品牌和商业分叉

<Warning>
Fork World Monitor、修改名称/Logo/品牌，并把它作为你自己的产品部署，在任何情况下都**不允许**，除非你拥有商业许可证。AGPL-3.0 要求：

1. **你必须保留源码和 UI 中所有原始版权声明**。
2. **你必须显著说明你的版本是 World Monitor 的修改分支**，并链接回原始项目。
3. **你必须以 AGPL-3.0 发布完整的修改后源码**，并对所有使用你部署的人可访问。
4. **未经维护者单独授予的商业许可证，不得用于商业用途**。

仅仅改名并不能消除这些义务。删除版权头也是另一项法律违规。
</Warning>

## 商业用途需要单独许可证

**任何商业用途都需要向维护者获取单独商业许可证。** 这包括但不限于：

- **重新品牌或白标**：fork 代码，改名/Logo，并把它作为你的产品或服务发布
- **作为付费产品运行**：通过付费墙、订阅或收费方式提供 World Monitor（或其修改版）
- **嵌入商业产品**：在你出售的产品里使用 World Monitor 的代码、组件、UI 或数据管线
- **SaaS 或咨询**：把 World Monitor 的能力作为咨询项目、托管服务或 SaaS 平台的一部分提供
- **企业内部使用**：在盈利性公司内部使用 World Monitor，以获取收入或竞争优势
- **出售由 World Monitor 派生的数据**：使用本平台收集、处理或转售情报数据以获取商业收益

AGPL-3.0 仅适用于非商业、个人、教育和研究用途。如果你的使用场景涉及任何商业活动，你必须联系维护者获取商业许可证。

**联系**：通过 [GitHub repository](https://github.com/koala73/worldmonitor) 或直接给维护者发邮件。

## 执行

维护者会主动监控未经授权的商业使用、重新品牌和许可证违规。如果你被发现违规：

1. 你会收到正式下架通知，要求立即停止侵权活动。
2. 如果违规没有及时解决，维护者保留依法寻求法律救济的权利，包括禁令和损害赔偿。
3. AGPL 违规可在国际版权法、DMCA 以及欧盟和其他司法辖区的同类法规下执行。

如果你知道有未经授权的商业分支或重新品牌部署，请通过 [GitHub Issues](https://github.com/koala73/worldmonitor/issues) 报告，或直接联系维护者。

## 常见场景

**个人使用或研究**：可自由使用、修改和自托管。只需遵守 AGPL 要求。

**非营利组织或教育机构自托管**：AGPL 允许。如果你修改了代码并公开可访问，你必须分享你的修改。

**运行修改过的公开实例（非商业）**：你必须保留所有版权声明，说明这是 World Monitor 的分支，并向与你的部署交互的用户提供修改后的源码。

**在你自己的工具中使用 World Monitor 的公共 API**：如果你的工具调用 World Monitor 的公共 API 并显示结果，那么你的工具并不构成衍生作品。AGPL 义务不适用。不过，如果你的工具是商业用途，你必须遵守 API 服务条款。

**提交 pull request**：提交 PR 就表示你同意你的贡献将以 AGPL-3.0 许可，与项目其余部分保持一致。

**商业使用（以上任一场景用于盈利目的）**：需要单独商业许可证。请通过 [GitHub repository](https://github.com/koala73/worldmonitor) 联系维护者。

## 为什么是 AGPL + 商业许可证

World Monitor 聚合的是公开可用的情报数据，并让所有人都能访问。双许可模式（开源部分用 AGPL，商业用途用商业许可证）能够确保：

1. **社区受益**：独立研究者、记者、学生和开源贡献者可以自由使用并改进平台。
2. **商业使用可持续**：从 World Monitor 获得商业价值的公司通过许可费回馈，从而资助持续开发。
3. **项目保持开放**：AGPL 防止商业实体拿走开源代码、加入专有功能，然后在不回馈项目的情况下与项目竞争。
