# Contributor Covenant 行为准则

## 我们的承诺

作为成员、贡献者和维护者，我们承诺让 World Monitor 社区成为一个对所有人都没有骚扰的环境，不论年龄、身体尺寸、可见或不可见的残障、族裔、性特征、性别认同与表达、经验水平、教育程度、社会经济地位、国籍、外貌、种族、种姓、肤色、宗教、性别认同或性取向如何。

我们承诺以有助于建立一个开放、友好、多元、包容且健康社区的方式来行动和互动。

## 我们的标准

有助于营造积极环境的行为示例包括：

* 对他人表现出同理心和善意
* 尊重不同的意见、观点和经历
* 提供并优雅地接受建设性反馈
* 对自己的错误承担责任，并向受影响的人道歉，同时从中吸取经验
* 关注的不只是个人利益，也关注整个社区的最佳利益

不可接受的行为示例包括：

* 使用性暗示语言或图像，以及任何形式的性骚扰或性挑逗
* 恶意挑衅、侮辱性或贬损性评论，以及个人攻击或政治攻击
* 公开或私下骚扰
* 未经明确许可，发布他人的私人信息，例如实际地址或电子邮件地址
* 其他在专业环境中通常会被视为不恰当的行为

## 适用范围

本行为准则适用于所有社区空间（GitHub issues、pull requests、讨论以及任何相关通信渠道），也适用于个人在公开场合正式代表本社区时。

## 执行

任何辱骂、骚扰或其他不可接受的行为，都可以通过 **[GitHub Issues](https://github.com/koala73/worldmonitor/issues)** 向项目维护者报告，或者直接通过 GitHub 联系仓库所有者。

所有投诉都会被及时、公正地审查和调查。项目团队有义务对举报人的身份信息保密。

## 执行指南

社区领导者会根据以下社区影响指南来判断某项行为违反本行为准则后的后果：

### 1. 纠正

**社区影响**：使用了不恰当的语言，或其他在社区中被视为不专业或不受欢迎的行为。

**后果**：由社区领导者私下发送书面警告，说明违规性质并解释为什么该行为不恰当。可能会要求公开道歉。

### 2. 警告

**社区影响**：单次事件或一系列行为造成的违规。

**后果**：发出带有后续行为限制的警告。在指定时间内，不得与相关人员进行任何互动，包括未经请求的接触，也包括在社区空间以及社交媒体等外部渠道中的互动。违反这些条款可能导致临时或永久封禁。

### 3. 临时封禁

**社区影响**：严重违反社区标准，包括持续的不当行为。

**后果**：在指定时间内，禁止与社区进行任何形式的互动或公开交流。在此期间，不允许与相关人员进行任何公开或私下互动，包括未经请求的接触。如果违反这些条款，可能会导致永久封禁。

### 4. 永久封禁

**社区影响**：持续违反社区标准，包括长期不当行为、骚扰个人，或者对某类人群进行攻击或贬损。

**后果**：永久禁止以任何形式在社区内进行公开互动。
